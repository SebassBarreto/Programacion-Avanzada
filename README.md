# Programacion-Avanzada — ¿qué es este repositorio y qué usa?

## ¿Qué es este repositorio?
**`SebassBarreto/Programacion-Avanzada`** es un repositorio orientado a prácticas/ejemplos de la asignatura **Programación Avanzada**, organizado por temas y ejercicios.

### Estructura principal
- `Archivos` — ejercicios relacionados con manejo de archivos.
- `BD` — contenido relacionado con bases de datos.
- `Hilos` — ejemplos de concurrencia/multihilo.
- `Microservicios` — ejemplos y proyectos de microservicios y consumo de servicios.
- `Sockets` — ejercicios de comunicación por sockets.
- `Talleres-Parciales` — talleres / parciales / prácticas evaluables.

---

## Resumen
- **Dominante:** Java, con apoyo de HTML/CSS/JS para ejemplos de frontend.
- **Temas:** Peristencia de Objetos, Programacion Concurrente, Programacion en red, Arquitectura orientada a microservicios, y Programacion Web.
